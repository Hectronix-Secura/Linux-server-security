# Linux Server Security & Hardening

## Overview
This project provides a practical checklist and basic scripts to secure a Linux server.
It is designed for small businesses, VPS owners and individuals who want a safer Linux environment.

## What is included
- SSH hardening
- Firewall configuration (UFW)
- System updates and patching
- User and permission review
- Basic security audit checklist
- Log monitoring basics

## Tools & Technologies
- Linux (Ubuntu, Debian, Kali)
- Bash
- UFW
- SSH
- System utilities

## Security Tasks Covered
- Disable root login
- Change default SSH port
- Configure SSH key authentication
- Enable firewall rules
- Remove unnecessary services
- Secure file permissions
- System update automation

## How to use
1. Review the checklist before applying changes
2. Apply configurations step by step
3. Always test on a non-production server first

## Disclaimer
This project is for educational and professional hardening purposes only.
Always make backups before applying security changes.
